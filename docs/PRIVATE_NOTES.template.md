# Private Notes Template

Use this file as a starting point for notes that should stay out of the public repository.

Copy this file to:

```text
docs/PRIVATE_NOTES.md
```

That file is ignored by git.

## Suggested Sections

### Personal Deployment Targets

- NAS hostnames
- Tailscale addresses
- Internal LAN IPs
- Reverse proxy notes

### Personal Runbook

- SSH steps
- SMB paths
- local backup locations
- manual recovery steps

### Non-Public Decisions

- Which friends are using which deployment
- private issue notes
- unpublished experiments

### Secrets That Still Must Not Be Committed

- API keys
- tokens
- passwords
- session exports
